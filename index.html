FreeBSD Ports

I mainly maintain following ports. To see information of my ports, visit FreshPorts. Some of my experimental ports haven't been included in official ports tree yet. See also my redports repository.

  • TigerVNC - High-performance, platform-neutral implementation of VNC
  • xrdp - Open source remote desktop protocol(rdp) server
  • x11rdp - X11 server for xrdp

Checking out my redports repository

x11rdp is still unofficial port. If you interested in x11rdp, please follow this instruction.

$ svn checkout https://svn.redports.org/meta

$ cd x11-servers/x11rdp # x11rdp for xrdp-0.6.x, x11rdp-devel for xrdp-0.7.x

# make install
